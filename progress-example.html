jQuery Progress Plugin

This is a jQuery Plugin displays the progress of an action until it is completed.

You can change any of the following options to customize it to fit your needs:


// options
$.uiProgress({
  title: 'Progress', // dialog title
  content: 'Please wait...', // dialog content string or html
  continuous: false, // continuously poll even if 'finished'
  modal: true, // modal dialog?
  pollTimer: 5, // how often plugin should poll, in seconds
  ageTimeout: 3, // reload page if JSON data age exceeds this value, in milliseconds
  url: null // JSON request URL, required
});

NOTE: The URL should returns JSON data with the following properties:

  • "age" - Age of the request, in milliseconds. Used to determine if request has timed out
  • "finished" - Boolean value that represents if action is complete
  • "max" - Maximum value of the progress bar (see: max)
  • "value" - The value of the progress bar (see: value)
